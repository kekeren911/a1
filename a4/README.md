# AI 居家生活助手

一个现代化、美观的 AI 助手界面，帮助用户解决日常生活中的各种问题。

## 功能特点

- 🎨 **现代化界面设计** - 渐变配色、流畅动画、响应式布局
- 💬 **智能对话系统** - 支持日常问题的智能响应
- ⚡ **快捷问题按钮** - 一键发送常用问题
- 🍳 **生活指引生成** - 食谱推荐、清洁技巧、收纳建议等
- 📱 **移动端适配** - 完美支持各种屏幕尺寸

## 支持的问题类型

1. **烹饪美食** - 菜谱推荐、烹饪技巧、食材搭配
2. **清洁技巧** - 油污去除、家居清洁、污渍处理
3. **收纳整理** - 衣物收纳、空间利用、断舍离建议
4. **购物管理** - 购物清单、物品采购、生活用品
5. **生活妙招** - 各种实用生活小技巧

## 使用方法

直接在浏览器中打开 `index.html` 文件即可使用。

- 输入框中输入你的问题
- 或点击快捷问题按钮
- 按 Enter 发送消息

## 文件说明

- `index.html` - 主页面结构
- `styles.css` - 样式文件
- `app.js` - 交互逻辑和 AI 响应系统

## 技术栈

- 纯 HTML5 / CSS3 / JavaScript
- 无需依赖任何第三方库
- 完全本地运行，数据安全
