# VR 星空馆沉浸式交互场景

一个基于 Three.js 的 360° 球形星空背景 VR 体验，模拟银河、星座、行星运转，支持 VR 模式和键鼠操作模式。

## 功能特性

### 🌌 核心功能
- **360° 球形星空背景** - 10000+ 星星点阵
- **银河模拟** - 螺旋状星云效果
- **行星系统** - 6大行星（水星、金星、地球、火星、木星、土星）
- **真实轨道** - 按比例缩行的行星轨道
- **土星环** - 特殊的行星环效果

### 🎮 交互功能
- **VR模式** - 适配 VR 设备（需要支持 WebXR）
- **键鼠模式** - 拖动旋转、滚轮缩放
- **行星点击** - 显示行星详细信息
- **自动旋转** - 可调节的宇宙旋转速度
- **行星动画** - 点击行星触发旋转展示

### ✨ 视觉效果
- **星际粒子** - 2000+ 青色粒子特效
- **发光效果** - 行星自发光和反射光
- **动态光源** - 太阳作为点光源
- **渐变背景** - 深空色彩渐变
- **毛玻璃UI** - 现代化半透明界面

## 技术栈

- **Three.js** - 3D 图形渲染
- **WebXR** - VR 设备支持
- **OrbitControls** - 相机控制
- **ES6 Modules** - 现代 JavaScript
- **CSS3** - UI 动画和样式

## 快速开始

### 1. 本地运行
```bash
# 直接打开 index.html
open index.html
```

### 2. 使用本地服务器
```bash
# 使用 Python 3
python -m http.server 8000
# 然后访问 http://localhost:8000

# 使用 Node.js
npx http-server
```

### 3. 体验指南

#### VR 模式
- 点击 "进入 VR" 按钮
- 需要 WebXR 兼容的 VR 设备（如 Meta Quest、Pico 等）
- 支持手势控制和头部追踪

#### 键鼠模式
- **鼠标拖动** - 360° 旋转视角
- **滚轮** - 缩放视图（5-50单位）
- **点击行星** - 查看详细信息
- **自动旋转** - 宇宙背景自动旋转
- **速度调节** - x1/x2/x5 三档速度

## 开发说明

### 扩展功能建议
1. **添加更多行星** - 天王星、海王星等
2. **星座连线** - 显示星座图案
3. **时间控制** - 加速/减速时间流逝
4. **音效系统** - 背景音乐和交互音效
5. **更真实的光照** - 使用物理渲染（PBR）
6. **陨石动画** - 流星划过效果
7. **空间站模型** - 可交互的空间站

### 性能优化
- 使用 InstancedMesh 优化大量粒子
- LOD（Level of Detail）系统
- 延迟加载和纹理压缩
