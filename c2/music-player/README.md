# 🎵 音乐流媒体播放器

一个功能完整的音乐流媒体播放器 UI，带有播放列表管理系统、波形可视化、歌词同步和离线支持。

## ✨ 主要功能

### 1. 🎚️ 音频播放器控制
- **播放/暂停**: 点击中央按钮控制播放
- **上一首/下一首**: 在播放列表中导航
- **进度条**: 拖动进度条跳转到特定位置
- **音量控制**: 独立的音量滑块控制
- **随机播放**: 随机打乱播放顺序
- **循环播放**: 重复播放当前歌曲

### 2. 📊 波形可视化
- 使用 **Wavesurfer.js** 实现实时波形显示
- 显示播放进度和音频频率
- 可交互：点击波形跳转到指定位置
- 平滑动画效果

### 3. 🔍 搜索功能
- **自动完成建议**: 输入时实时显示匹配歌曲
- 支持按歌曲名称和艺术家搜索
- 即时点击播放搜索结果
- 最多显示 5 个建议结果

### 4. 🎤 时间同步歌词
- 根据播放时间自动高亮当前歌词
- 自动滚动保持当前歌词可见
- 平滑过渡动画
- 支持多首歌曲歌词

### 5. 📋 播放列表管理
- 显示完整播放列表
- 当前播放歌曲高亮显示
- 点击任意歌曲立即播放
- 添加歌曲到播放队列
- 从队列中移除歌曲

### 6. 👥 协作播放列表模拟
- 模拟多人实时协作
- 每 15 秒自动更新播放列表
- 随机添加新歌曲
- 随机重排播放列表
- 显示协作状态指示器

### 7. 📱 离线播放支持
- 使用 **Cache API** 缓存音频文件
- 自动缓存前 5 首歌曲
- 播放时自动缓存新歌曲
- 显示离线状态
- 网络断开时播放已缓存歌曲

### 8. 📐 响应式设计
- 完美适配手机、平板和桌面设备
- 设备旋转时自动调整布局
- **旋转时不中断播放**
- 触摸屏友好的控制界面
- 自适应网格布局

## 🚀 使用方法

### 本地运行

1. 克隆或下载项目文件
2. 确保所有文件在同一目录下：
```
music-player/
├── index.html
├── styles.css
├── app.js
└── README.md
```

3. 使用本地服务器打开（推荐）：
```bash
# Python 3
python -m http.server 8000

# Node.js (使用 serve)
npx serve .

# 或使用 VS Code Live Server 扩展
```

4. 在浏览器中打开 `http://localhost:8000`

### 直接打开
由于使用远程音频文件，也可以直接在浏览器中打开 `index.html`。

## 🎮 操作指南

### 基本播放
1. 点击播放列表中的任意歌曲开始播放
2. 使用播放控制按钮控制播放
3. 拖动进度条跳转位置

### 搜索歌曲
1. 在搜索框输入歌曲名或艺术家名
2. 从下拉建议中选择歌曲
3. 点击即可播放

### 管理播放队列
1. 点击歌曲旁的 `+` 按钮添加到队列
2. 队列中歌曲按顺序播放
3. 点击 `✕` 按钮从队列中移除

### 随机和循环
- 点击随机按钮启用随机播放
- 点击循环按钮启用单曲循环
- 按钮激活时会高亮显示

## 🛠️ 技术栈

- **HTML5**: 语义化结构

- **JavaScript ES6+**: 核心功能实现

## 🎨 自定义

### 修改颜色主题
在 `styles.css` 中修改 CSS 变量：
```css
:root {
    --primary-color: #1db954;  /* 主色 */
    --accent-color: #1ed760;   /* 强调色 */
    --bg-dark: #121212;        /* 深色背景 */
    /* ... */
}
```

### 添加自定义歌曲
在 `app.js` 中修改 `MOCK_SONGS` 数组：
```javascript
const MOCK_SONGS = [
    {
        id: 1,
        title: "歌曲名",
        artist: "艺术家",
        duration: 245,
        url: "音频URL",
        cover: "渐变色或图片URL"
    },
    // 添加更多歌曲...
];
```

### 添加自定义歌词
在 `app.js` 中修改 `MOCK_LYRICS` 对象：
```javascript
const MOCK_LYRICS = {
    1: [ // 对应歌曲ID
        { time: 0, text: "♪ 前奏 ♪" },
        { time: 15, text: "第一句歌词" },
        // 添加更多歌词行...
    ]
};
```
